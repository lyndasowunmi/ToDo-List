 ToDo List is a simple web/mobile application to save your daily tasks in order to stay organized.

## Features:
* Add new task
* Edit task
* Delete single/all tasks
* Mobile responsive
* Local storage backup

### Tools:
* CSS
* JavaScript
* React
* TypeScript
